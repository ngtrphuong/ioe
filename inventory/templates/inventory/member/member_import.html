CSV File Format Requirements

Please prepare your CSV file according to the format below to ensure data accuracy.

CSV file must include the following columns:

  • name (required): Member Name
  • phone (required): Phone number, must be 11 digits
  • email: Email address
  • member_id: Member ID, will be auto-generated if not provided
  • level: Member level name, will use default level if not provided or not found
  • points: Member points, must be an integer
  • birthday: Birthday, format: YYYY-MM-DD
  • address: Member address
